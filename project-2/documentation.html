Noteworthy:

  • Bulma Framework:
    • Very mobile friendly and very responsive. All elements scale properly to the screen size and a hamburger menu icon pops up to replace the navbar when the screen size gets small enough for mobile devices.
    • Utilizes different layout classes to help structure the page such as:
      1. Tiles which allows the user to map out their elements
      2. Section classes which are used to emphasize a section of elements by adding an indent and shifting everything to the left.
      3. Content classes which are used to structure the effected elements like a normal HTML page

Grading:

  • Meeting Requirements: I met all requirements by basing a good chunk of my effects off of a dynamical system (Periodic Functions & Perlin Noise), gave well over 3 controls for the user to play around with, and I implemented presets using fetch API for the user to explore what the app is capable of.
  • Above and Beyond: I went above and beyond through exploring the different effects I can create utilizing different trig functions. The electricity effect is based off of a sine wave, can have different styles based on the angle, and its amplitude is based on audio data. Furthermore, I created the wave effect which rotates around the center of the canvas and utilizes 4 different trig functions for different wave styles. All effects are in at least one way affected by audio data and can easily be edited via the controls under the canvas.
