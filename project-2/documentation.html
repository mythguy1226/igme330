    • Utilizes different layout classes to help structure the page such as:
      1. Tiles which allows the user to map out their elements
      2. Section classes which are used to emphasize a section of elements by adding an indent and shifting everything to the left.
      3. Content classes which are used to structure the effected elements like a normal HTML page
